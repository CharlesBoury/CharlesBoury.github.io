Bact Attackbacteria fighting

I made the graphics for a game commissioned to raise awareness about Antibiotic Resistance.

It’s an endless Tetris-like arcade mobile game about bacterias. Fight them with antibiotics but don’t use the wrong one ! Otherwise they’ll end up stronger.

Roles:
Artistic Direction, 2D assets production, Animations & Unity integration.

Artworks by Charles Boury, 2017.
